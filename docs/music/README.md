# Music

This directory contains documentation about the musical content of the project:
the song catalog, arrangement experiments, and insights about what works.

---

## Song catalog

`swedish_children_songs.csv` in the root of the repository is the canonical list
of songs. It tracks the title, origin, copyright status, and where to find the
source files (MIDI, sheet music) for each song.

Source files for each song live on Google Drive. See [../drive](../drive/README.md)
for how Drive is organized.

---

## Arrangement experiments

Arrangement files (MusicXML iterations, MIDI exports) are stored under `assets/xml/`
and `assets/midi/` locally, and on Drive for sharing.

When experimenting with a new arrangement, use a descriptive filename that records
what was tried:

```
002_Bjornen_sover_arranged.musicxml
002_Bjornen_sover_hiphop.musicxml
002_Bjornen_sover_hiphop_swagger.musicxml
```

---

## Insights

Document what you learn from experiments here. Each insight should say:
- what was tried
- what worked or did not work
- any rule of thumb that could apply to future arrangements

This does not need to be formal. Short notes are fine.
